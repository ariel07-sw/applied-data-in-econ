# Angrist & Evans (1998) — IV / 2SLS Replication

## Data Source
AngEv98.zip from Joshua Angrist’s MIT Data Archive (1980 & 1990 Census extracts).

## Repo Structure
- `data/raw/`: original files (excluded from git)
- `data/processed/`: processed outputs (excluded from git)
- `notebooks/01_Data_Cleaning.ipynb`: SAS -> CSV
- `notebooks/02_Replication.ipynb`: 2SLS replication

## How to Run
1. Put the downloaded `.sas7bdat` files into `data/raw/`
2. Run `01_Data_Cleaning.ipynb`
3. Run `02_Replication.ipynb`

## Note
Data files are not uploaded to GitHub due to size limits.
